
# My Dotfiles

For my Ubuntu, but also has vscode stuff

## Requirements

 - git
 - gnu stow
 


## Download

- download all the zsh plugins
- download tpm
- <prefix>+I (prefix + capital i) to install tmux stuff within tmux

```zsh
  git clone https://github.com/jakobevangelista/dotfiles.git
  cd dotfiles
  stow .
```
